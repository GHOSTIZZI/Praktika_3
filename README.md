Описание проекта

Музыкальное приложение на JavaFX с интеграцией PostgreSQL/Supabase для прослушивания треков, управления альбомами, плейлистами и радиостанциями.

Приложение позволяет:

просматривать треки;
искать исполнителей;
создавать личные плейлисты;
просматривать альбомы администратора;
слушать радио;
добавлять треки в избранное.


Функциональность


Пользователь:

регистрация и вход;
просмотр каталога треков;
поиск музыки;
создание личных плейлистов;
удаление только своих плейлистов.

Администратор:

создание альбомов;
управление featured playlists;
удаление альбомов главного меню;
добавление треков в альбомы.

Технологический стек:

Java 17
JavaFX
PostgreSQL
Supabase REST API
Maven
Git / GitHub

Установка и запуск:

Клонировать репозиторий.
Открыть проект в IntelliJ IDEA.
Установить зависимости Maven.
Запустить.
